# PhishGuard – Phishing URL Detection

This project is a machine learning-based web app that detects whether a URL is safe or phishing.

## Features

* URL analysis
* Machine learning prediction
* Basic threat detection

## Tech Used

* Python
* Streamlit
* scikit-learn
* pandas, numpy

## How to Run

1. Clone the repo:

```
git clone https://github.com/saurabh-krishnan/phishing-url-de_obfusctor.git
cd phishing-url-de_obfusctor
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run app.py
```
